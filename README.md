# Toon

A simple Travel App with a search bar to discover destinations.

## Features

- **Search Bar**: Search for destinations, cities, or attractions with real-time filtering
- **Responsive Design**: Works on desktop and mobile devices
- **Accessible**: Built with accessibility best practices (ARIA labels, semantic HTML)

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser

No build step required - this is a vanilla HTML/CSS/JavaScript application.

## Usage

Type in the search bar to filter destinations. The app searches through destination names and descriptions to find matches.

## Sample Destinations

The app includes sample destinations including Paris, Tokyo, New York, Barcelona, Sydney, Rome, London, Dubai, Bali, Cape Town, Machu Picchu, and Santorini.

## Files

- `index.html` - Main HTML structure
- `styles.css` - CSS styling
- `app.js` - JavaScript functionality
